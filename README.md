# 3D Puzzle Generator

A browser-based interactive 3D puzzle generator built with React, Three.js, and deployed on AWS Amplify.

## Puzzle Types
- **Jigsaw Puzzle** — Upload an image and solve it as a 3D jigsaw
- **Custom Geometry** — Explode a 3D shape into pieces and reassemble it

## Stack
- React 18 + TypeScript + Vite
- @react-three/fiber + @react-three/drei (Three.js)
- Tailwind CSS
- Vitest + Playwright (testing)
- AWS Amplify (deployment)

## Getting Started

```bash
npm install
npm run dev
```
